# M6-01 — Генератор демо-активности (боты-перевозчики)

**Майлстоун:** M6 (оживление)
**Зависимости:** M2-06, M3-03
**Оценка:** M (1 день)

## Цель
Фоновый генератор (§2 «Источник лотов» п.2): периодически создаёт синтетические лоты и эмулирует ставки ботов, чтобы realtime-демо не выглядела мёртвой. Попутно — ещё один продюсер/консьюмер на тему инфры.

## Объём работ
- Scheduled job (`@nestjs/schedule`): периодически `CreateLot` синтетических грузов.
- Боты-перевозчики: ставят ставки по открытым лотам через тот же горячий путь (`POST /lots/:id/bids` или внутренний вызов), реалистичная динамика (понижение цены, всплески к закрытию → проверка анти-снайпа).
- Конфиг включения/интенсивности через env (выключаемо для тестов).
- Корректное взаимодействие с rate-limit и идемпотентностью.

## Definition of Done
- При включённом генераторе список и страница лота «живут»: появляются лоты и ставки.
- Боты провоцируют анти-снайп-продления (видно на витрине).
- Генератор выключается флагом и не мешает тестам.
