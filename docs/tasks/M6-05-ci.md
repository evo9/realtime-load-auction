**Зависимости:** M1-02 (и тесты, накопленные по ходу)

## Цель
GitHub Actions pipeline (§12): lint + test + build. Интеграционные тесты на реальных Postgres/RMQ/Redis (testcontainers) — сильный сигнал, что инфра-паттерны проверены, а не задекларированы (§13).

## Объём работ
- Workflow: матрица jobs — lint (eslint/prettier), unit-test, integration-test (testcontainers PG/RMQ/Redis), build (api+web).
- Кэш зависимостей, прогон миграций перед integration-тестами.
- Бейдж статуса в README.
- (Опц.) job сборки docker-образа / деплой демо.

## Definition of Done
- CI зелёный на PR: lint+unit+integration+build проходят.
- Интеграционные тесты реально поднимают контейнеры и проверяют outbox/CAS/saga/retry-DLX.
- Бейдж в README.
