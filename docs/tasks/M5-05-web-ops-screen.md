# M5-05 — web: ops-экран (саги + DLQ) — опционально

**Майлстоун:** M5 (фронт) — опционально, сильный сигнал
**Зависимости:** M5-01, M4-04
**Оценка:** S (0.5 дня)

## Цель
Ops-экран (§11): визуализация состояний саг и содержимого DLQ. Опционально, но это заметный сигнал «понимаю эксплуатацию распределённых систем».

## Объём работ
- Страница `/ops`: таблица саг (`GET /ops/sagas`) — шаг, статус, лот, попытки.
- Панель DLQ (`GET /ops/dlq`) — очередь, причина, счётчик.
- (Опц.) кнопка requeue/повтор за флагом.
- Доступ за ролью/флагом.

## Definition of Done
- Видны running/failed/compensated саги и сообщения DLQ.
- Экран читаем и пригоден для демонстрации на собесе.
