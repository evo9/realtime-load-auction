# M5-04 — web: дашборд «Мои ставки»

**Майлстоун:** M5 (фронт)
**Зависимости:** M5-01, M3-04
**Оценка:** S (0.5 дня)

## Цель
Дашборд перевозчика (§11): ставки текущего carrier и их статус.

## Объём работ
- Страница `/me/bids` (`GET /me/bids`).
- Статус по каждой ставке: лидирует / перебита / выиграна / лот закрыт.
- Live-обновление статуса через WS (опц., если открыт соответствующий лот).
- Ссылки на лоты.

## Definition of Done
- Перевозчик видит свои ставки и актуальный статус.
- Данные обновляются после новой ставки/закрытия.
