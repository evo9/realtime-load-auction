# M5-02 — web: список лотов (SSR + фильтры)

**Майлстоун:** M5 (фронт)
**Зависимости:** M5-01, M2-07
**Оценка:** S (0.5 дня)

## Цель
Страница списка лотов (§11): SSR для быстрого first paint, простые фильтры поверх listing read-model.

## Объём работ
- SSR-страница `/lots` (`GET /lots`), быстрый first paint.
- Фильтры: статус, equipmentType, маршрут (origin/destination).
- Карточка лота: маршрут, окно, текущая лучшая ставка, обратный отсчёт до closeAt, ссылка на страницу лота.
- Пагинация/курсор.

## Definition of Done
- Список рендерится на сервере, фильтры меняют выдачу.
- Переход на страницу живого лота работает.
