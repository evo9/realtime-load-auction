# M2-08 — Seed: шипперы, пользователи, реалистичные лоты

**Майлстоун:** M2 (лоты)
**Зависимости:** M2-06, M1-05
**Оценка:** S (0.5 дня)

## Цель
Стартовый набор данных (§2 «Источник лотов» п.1): несколько шипперов/перевозчиков и реалистичные грузы, чтобы демо не было пустым. Генератор активности — отдельно (M6-01).

## Объём работ
- `apps/api/seed`: сидер пользователей (`shipper`×N, `carrier`×N) с известными паролями для демо.
- Сидер реалистичных лотов (разные маршруты, equipmentType, окна `openAt/closeAt`), часть — со скорым открытием для демо.
- Подключить к `make seed` (из M1-01).
- Идемпотентность сидера (повторный запуск не плодит дубли).

## Definition of Done
- `make seed` наполняет БД; `GET /lots` отдаёт лоты; логин сид-юзеров работает.
- Повторный `make seed` не создаёт дубликаты.
